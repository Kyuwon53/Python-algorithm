# 🔔 프로그래머스 - Level 1
## 📑 핸드폰 번호 가리기
### 📌 문제 설명
 전화번호가 문자열 phone_number로 주어졌을 때, 전화번호의 뒷 4자리를 제외한 나머지 숫자를 전부 *으로 가린 문자열을 리턴하는 함수
### ✔ 제한 사항
- s는 길이 4 이상, 20이하인 문자열입니다.

### 💡 아이디어
1. 입력 값의 길이에서 4를 뺀 길이만큼 *로 바꾼다 
2. 정규식을 사용해서 뒤에 4자리만 남기고 다 *로 변환
