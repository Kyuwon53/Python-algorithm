# 🔔 프로그래머스 - Level 1
## 📑 가운데 글자 가져오기

### 📌 문제 설명
단어 s의 가운데 글자를 반환하는 함수, solution을 만들어 보세요. 
단어의 길이가 짝수라면 가운데 두글자를 반환하면 됩니다.

### ✔️ 제한 사항
- s는 길이가 1 이상, 100이하인 스트링입니다.

### 💡 아이디어
- 길이 짝수, 홀수 분기문
- 홀수면 길이/2 번째 글자 반환
- 짝수면 길이/2, 길이/2+1 번째 글자 반환

### 💬 개선사항
- 분기문을 작성하지 않고 로직을 생각하자
- return str[(len(str)-1)//2:len(str)//2+1]
- `//`는 몫을 반환
